Attendance System

Description: The Attendance Monitoring System for Buting Senior High School is an innovative solution designed to enhance the efficiency and accuracy of attendance tracking using QR codes. This system caters to four types of users: teachers, parents, students, and administrators. By leveraging real-time data and automated notifications, the system aims to improve overall attendance monitoring and communication within the school community.

Key Features:

  • QR Code Scanning: Students use QR codes for quick and accurate attendance marking, reducing manual entry errors.
  • Real-Time Parent Notifications: Every time a QR code is scanned, an automatic email is sent to parents, providing instant updates on their child's attendance status.
  • Weekly Attendance Reports: Parents receive a weekly summary report of their child's attendance, allowing them to track performance over time.
  • Teacher Tools: Teachers can generate School Form 2 (SF2), required by DepEd, directly from the system. Additionally, they can make announcements, assign homework, and contact parents as needed.
  • Admin Dashboard: Administrators have access to comprehensive tools for managing users, viewing overall attendance statistics, and maintaining the system.
  • User-Friendly Interface: The system is designed with an intuitive interface for easy navigation and use by all user types.

Technologies Used:

  • HTML
  • CSS
  • JavaScript
  • Bootstrap
  • SQL
  • PHP

QuizQuest

Description: QuizQuest is an engaging web application designed to offer an interactive and secure quiz-taking experience. This platform allows users to participate in quizzes on various topics, while providing administrators with the tools to create, manage, and customize quizzes. With a focus on security, QuizQuest ensures the integrity of the quiz experience by implementing strict measures that prevent users from refreshing or exiting the quiz portal. If a user attempts to leave or refresh the quiz, it will automatically end, regardless of completion status.

Key Features:

  • Interactive Quizzes: Users can take quizzes on a variety of topics with an intuitive interface that makes the experience enjoyable and educational.
  • Admin Tools: Administrators can create and manage quizzes, customize questions, and track user performance with a comprehensive backend dashboard.
  • Security Measures: Strict security protocols ensure that users cannot refresh or exit the quiz portal without ending the quiz, preserving the accuracy of results.
  • Responsive Design: Optimized for both desktop and mobile devices, providing a seamless experience across all platforms.
  • Real-Time Feedback: Instant feedback on quiz performance, helping users learn and improve.

Technologies Used:

  • HTML
  • CSS
  • JavaScript
  • Bootstrap
  • SQL
